# Assignote

Assignote is a Python desktop application for tracking assignment and tests.

## Features

-Add assignments and tests (modules), including their due dates
-View saved module data
-Delete individual assignments
-Wipe module data from the system completely
-Sort modules by due date, ranging from most urgent to least urgent for convenience
-Store data using JSON
-Display information using .txt

## Technologies

-Python
-JSON
-txt
